This folder is the root folder, and all folders use relative filepaths to read/write within it.

data_minimized (not in the github repo) contains all the data used in the analyses, and is broken up into multiple levels of processing.

src contains all the code used to execute the analyses broken down by analysis- these are all R files and Blender files.

figs contains the figures and tables included in the manuscript and supplemental materials that were generated using code.
